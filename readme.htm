0、安裝與開啟

系統平台：任何可執行Google Chrome 或 Safari 的作業系統。

網頁瀏覽器：建議使用Google Chrome、Safari。

安裝移除：搜尋引擎Accelon軟體無需安裝，複製資料夾「Yinshun」到本機磁碟即可使用。欲移除時，直接刪除資料夾「Yinshun」即可。

開啟執行：1、雙擊資料夾「Yinshun」中的檔名index.html。

2、點選畫面左上角「開啟本機資料庫」。

3、同時選取yinshun.cor、mpps.cor、taixu.cor、taisho.cor四種資料庫，點選「開啟」。

一、檢索方式與結果摘要

1、選取資料庫後，依需求範圍，點選「全選」、「全不選」或單一書（篇）名。

2、如有數個檢索詞，中間以「半形空格」分隔。如：「緣起 中道」。

3、當檢索詞超過15個字時，系統會自動開啟模糊檢索，依照關聯性比例呈現搜尋結果。

4、輸入檢索詞，點選「搜尋」，即顯示結果摘要。可選擇摘要行數，以利抉擇所需內容。點選其中一筆看本文之後，欲再看其他筆內容時，點選工具列「摘要」，即可跳回結果摘要畫面。

二、原文顯示與引文對應

1、進入單一書（篇）名或檢索摘要結果，瀏覽畫面將分為左右兩欄。左欄為起始欄；當以左欄為原文顯示欄時，右欄為引文對應欄。游標移至文字上方數字，即顯示引文出處或註解。

2、左欄文字下方顯示藍色虛線者，表示「連結功能」（需勾選☰工具盒中的「互文」選項）。點選後，結果顯示在右欄。

三、科判連結與交互參照

1、點選進入單一書（篇）名，文章中以黑色字體表示原文，以彩色字體表示科判。

2、點選《大智度論講義》左欄科判，可由左欄經文連結右欄論文；點擊右欄科判左上方數字，亦可由右欄論文連結左欄經文，左右欄交互參照。

四、「☰」工具盒

1、點選左欄左上角的☰圖示，出現小視窗。如欲取消工具盒內之個別選項，請取消打勾，所指定選項的內容就會被隱藏。

2、右欄輔助視窗顯示大正藏時，點選左上角的☰圖示，可以輸入大正藏頁碼，直接調閱指定頁數的內容。

五、內文關聯查詢（以《印順法師佛學著作集》為例）

1、左欄為《印順法師佛學著作集》內文。右欄上方為「網絡連線查詢」，由左而右依序為「Ｇoogle」、「一行」、「萌典」、「康熙字典」。右欄下方為《CBETA 電子佛典系列》內容。
選取內文欲查詢的文字，右欄「網絡連線查詢」上方則會顯示選取的文字→點選查詢工具→點擊查詢上方的文字，即自動執行網絡查詢。

2、點擊右欄CBETA藍色虛線的內容，會跳出「逆向連結」視窗，顯示印順法師著作中引用該內容的書名與頁碼→點擊頁碼，左欄畫面即自動連結至相關內容。

六、引用複製

選取欲引用複製的文字→點擊滑鼠右鍵→點選「複製」→在欲貼上已複製內容的軟體中（如MS Word）點擊滑鼠右鍵→點選「貼上」，即可顯示所選取的文字，並附記所引的書名及頁碼。

※常見問題

Q1：為何每次開啟都要「開啟本機資料庫，選擇yinshun.cor、taisho.cor、mpps.cor、taixu.cor」。不能換成一次作業嗎？這跟安全性有什麼關係？

A1：這是W3C規定的，所有瀏覽器都必須遵守，以免讓病毒有可趁之機。選檔的動作是必要的，這是保護使用者的安全限制，瀏覽器執行的javascript只能讀取使用者手動選擇的檔案，其他任何檔案（除了html和javascript）都無法讀取。

Q2：輸入檢索詞，顯示檢索所得摘要，點選其中一筆看本文之後，按「回上頁」時為何會跳離檢索畫面？

A2：File 協議不能記錄url history，所以不能back。請按「摘要」，即可跳回檢索所得畫面。

Q3：複製內容範圍比較多時，引用複製時沒有顯示書名及頁數，該如何處理？

A3：連按兩下Ctrl+C就可以正確顯示。

Q4：為甚麼引用複製小範圍內容時，沒有顯示書名及頁數？

A4：八個字以下只是單純複製，複製九個字或以上，才會顯示書名及頁數。

Q5：為甚麼輸入頁碼後不能調閱該頁內容？

A5：由於序與內文皆從第一頁開始，為了避免混淆，分為兩個檔案。請點入書名右側的「下卷」，當視窗顯示內文時，於頁碼處輸入數字（半形），視窗將轉換至含有該頁的章節；再按一次鍵盤Enter鍵，就會顯示指定頁數的內容。

Q6：為甚麼有一些頁碼沒有顯示出來？

A6：因為有些圖表跨頁，覆蓋了頁碼。如果要顯示全部頁碼，可以點擊左上角☰工具盒圖示，取消勾選「圖」與「表」。

Q7：為甚麼有些地方連結《CBETA 電子佛典系列》時，沒有出現經文？

A7：此問題主要是由於本光碟的CBETA資料庫未包含《大正藏》第56冊以後的內容、夾註與校勘，如果印順法師著作引用到這部分的內容，便無法超連結。

Q8：連結《CBETA 電子佛典系列》時，若未顯示正確經文時，如何處理？

A8：請記下相關內容並寄至yinshun.tw@msa.hinet.net。
